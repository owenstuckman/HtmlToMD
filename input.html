About This Page

This is a simple HTML file created to demonstrate basic web page structure and elements.

It includes a header, a main content area with a section, and a footer.

Key Elements Used

  • <!DOCTYPE html>: Declares the document type as HTML5.
  • <html>: The root element of an HTML page.
  • <head>: Contains meta-information about the HTML document (e.g., character set, viewport settings, title).
  • <body>: Contains the visible page content.
  • <header>, <main>, <section>, <footer>: Semantic elements for structuring content.
  • <h1>, <h2>: Heading elements.
  • <p>: Paragraph element.
  • <ul>, <li>: Unordered list and list item elements.
